# ♟️ Chess App Repository

This repository contains the user-facing Professional Chess Application, featuring:
1. **`client/`**: A high-performance Ionic React + Capacitor application supporting iOS, Android, and Web.
2. **`server/`**: A real-time Node.js Express + Socket.io server using Redis for horizontal scalability.

## 🚀 Quick Start
Refer to the `.env.example` in the root workspace folder to configure your environment variables.

### 💻 Run Server:
```bash
cd server
npm install
npm run dev
```

### 📱 Run Client:
```bash
cd client
npm install
npm run dev
```
